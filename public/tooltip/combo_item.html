Create Combo Menu Items

GMI - Group Menu Item allows you to create combo meals. The first item selected will be the main dish and that price will be listed as the price for the combo meal. Diner will be guided to pick each item included in the GMI group.
PMI - Paired Menu Item allows you to create combo meals. The first item selected will be the main dish and that price will be listed as the price for the combo meal. When Diner orders the main dish, the other items will be added to the order.
