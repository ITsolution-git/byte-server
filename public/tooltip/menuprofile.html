Create Menu Profile

Menu Profile Name - Profile that allow you to create new menu. Ex: Breakfast, Lunch, Happy Hour, Special
Menu Points - Customer receives points when they rate items on this menu 1.1
Restaurant Trigger - Restaurant receives a message when the diner rates a menu item at the value set or below. Messages are displayed in Message Center
This is the menu that your customers will see published on BYTE. Only one Menu Profile may be published as a time.
